# Guess the Word Game

A simple yet engaging word-guessing game implemented in Python with comprehensive testing using pytest.

## Overview

This is a classic word-guessing game where players try to guess a hidden word one letter at a time. The game provides feedback on each guess and tracks the number of attempts remaining. Built with clean, object-oriented Python code and thoroughly tested with pytest.

## How to Play

1. **Start the game:**

2. **Follow the prompts:**
   - The game will display underscores representing the hidden word
   - Guess one letter at a time
   - Correct guesses reveal the letter's position(s)
   - Incorrect guesses reduce your remaining attempts
   - Win by guessing all letters before running out of attempts!
